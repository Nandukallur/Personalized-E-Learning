PHP Programming Questions

  1. Palindrome Checker: Write a PHP function that takes a string as input and checks whether it is a palindrome or not. A palindrome is a word, phrase, or sequence of characters that reads the same backward as forward (ignoring spaces, punctuation, and capitalization).
  2. Write a PHP program that prints the numbers from 1 to 100. But for multiples of three, print "Fizz" instead of the number, and for the multiples of five, print "Buzz". For numbers that are multiples of both three and five, print "FizzBuzz".
  3. Write a PHP script that reads the contents of a text file and outputs the number of lines, words, and characters in the file. You can use any text file for testing.
  4. Factorial Calculator: Write a PHP function to calculate the factorial of a given non-negative integer.
  5. Write a PHP function that checks if a given number is a prime number.
  6. Write a PHP function to reverse a string without using the strrev() function.
  7. Write a PHP function that takes an array of numbers as input and calculates the sum and average of the numbers.
  8. Write a PHP function that checks if a given year is a leap year.
  9. Write a PHP function that takes an array as input and returns a new array containing only the unique elements.
  10. Write a PHP function that parses a URL into its components (scheme, host, path, query parameters, etc.).
